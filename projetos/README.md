# Projetos Finais

Esta pasta reúne os projetos finais publicados pelos alunos da turma.

## O que é o Projeto Final?

O projeto final é uma análise de dados completa sobre um tema à sua escolha, demonstrando:

- Coleta ou obtenção de um dataset real
- Limpeza e transformação dos dados com Pandas
- Análise exploratória (EDA)
- Visualizações claras e bem documentadas
- Conclusões baseadas nos dados

## Como publicar seu projeto final

1. Desenvolva o projeto em `alunos/seu-nome/projetos/projeto_final/`
2. Quando aprovado pelo professor, ele será movido (ou linkado) aqui
3. Seu projeto ficará público nesta galeria da turma
