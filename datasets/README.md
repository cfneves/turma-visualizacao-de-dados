# Datasets Compartilhados

Datasets utilizados nas aulas e exercícios da turma.

## Regras para datasets

- Arquivos até **50 MB** podem ser adicionados diretamente aqui
- Arquivos maiores: suba no Google Drive / Kaggle e adicione o link abaixo
- Formatos aceitos: `.csv`, `.json`, `.xlsx`
- Arquivos comprimidos (`.zip`, `.gz`, `.parquet`) são ignorados pelo `.gitignore`

## Como adicionar um dataset

```bash
# Copie o arquivo para esta pasta
cp meu_dataset.csv datasets/

# Commit com o prefixo "data"
git add datasets/meu_dataset.csv
git commit -m "data: adiciona dataset nome_do_dado"
```
