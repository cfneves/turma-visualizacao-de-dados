# Exercícios

Esta pasta contém os enunciados dos exercícios propostos ao longo do curso.

## Como entregar

1. Crie uma subpasta dentro de `alunos/seu-nome/exercicios/exercicio_XX/`
2. Resolva o exercício no Jupyter Notebook
3. Garanta que o notebook roda sem erros (`Kernel > Restart & Run All`)
4. Abra um Pull Request seguindo o [guia de contribuição](../CONTRIBUTING.md)
